# Классификация текстов IMDB

Будем решать задачу классификации отзывов на фильмы позитивные/негативные. Попробуем разные способы и эмбеддинги, модель на основе TextCNN

Этапы предобработки:
1. Замена сокращений
2. Удаление знаков препинания *
3. Удаление чисел *
4. Удаление пробелов *
5. Токенизация
6. Лемматизация
